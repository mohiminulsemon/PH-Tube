Var: var is used to define a variable in javascript. It is a functional scope, means it cant be accessed outside the function. It has hoisting issues.
let: It is also used to define a variable in javascript. It is a block scope, means it cant be accessed outside the block. It doesn't have hoisting issues like var.
const: Const is used to declare variable that has fixed value. it cant be changed later.

Null : Null is a value that represents nothing. It is an object/variable that has no value.
Undefined : Undefined is absence of a value of a variables. If a varibles declared but not assigned a value, it returns undefined.

REST API: REST API is an interface for web services. It stands for Representational State Transfer. It uses HTTP methods like GET, POST, PUT, PATCH, DELETE operations on data/resources. It connects server to clients.
